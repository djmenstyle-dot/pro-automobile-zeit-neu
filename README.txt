PA – App-Icon & PWA Dateien

1) Im Repo einen Ordner anlegen:  /public/icons/
2) Diese Dateien dort rein kopieren:
   - alle PNGs aus diesem Paket
   - favicon.ico
3) In /public/ die Datei manifest.webmanifest ablegen (liegt in diesem Paket)
4) In index.html in <head> einfügen (Beispiel unten im Chat)

Dann: Vercel redeploy → am Handy „Zum Startbildschirm hinzufügen“.
